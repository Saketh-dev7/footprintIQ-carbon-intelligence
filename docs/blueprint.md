# **App Name**: FootprintIQ

## Core Features:

- Intelligent Footprint Assessment: A comprehensive multi-step flow capturing transport, food, energy, shopping, and waste data with real-time feedback.
- Carbon Intelligence Hub: High-fidelity analytics dashboard utilizing Recharts to visualize emission sources and monthly trends.
- AI Sustainability Advisor: Uses generative AI as a tool to analyze footprint data and generate personalized high-impact action plans and reduction goals.
- Conversational Sustainability Assistant: Interactive chat interface using AI as a tool to answer specific questions about lifestyle impacts and green alternatives.
- Impact Forecast Studio: A simulation engine that allows users to interactively model lifestyle changes and visualize projected annual carbon savings.
- Personal Sustainability Journey: Secure storage for assessment history and badge progress to track reduction efforts over time.
- Carbon Calculation Engine: Server-side logic to compute modular emission scores across diverse lifestyle categories with high precision.
- Carbon Action Plan Generator: Uses AI as a tool to generate a personalized 30-day carbon reduction roadmap based on user lifestyle data.
- EcoScore: A numerical indicator that evaluates sustainability performance on a scale of 0 to 100.
- Monthly Carbon Reduction Goal: An interactive interface for setting and tracking progress towards specific emission reduction targets.

## Style Guidelines:

- Primary Color: Vivid Emerald (#1BB49F), used for key sustainability metrics and CTA highlights.
- Background Color: Deep Forest Night (#0B1110), a rich, desaturated dark theme reflecting premium SaaS elegance.
- Accent Color: Spring Glade (#2BD05E), providing analogous contrast for success states and growth-related data.
- Headline font: 'Space Grotesk', a tech-forward sans-serif. Body font: 'Inter' for high readability in data-dense dashboard views.
- Minimalist, thin-stroke SVG icons featuring environmental and tech motifs for a refined, modern intelligence look.
- A sleek, bento-grid dashboard structure with glassmorphism cards and spacious padding for a clean intelligence profile.
- Sophisticated layout transitions using Framer Motion to glide between assessment steps and chart loading states.
